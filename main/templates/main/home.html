Welcome to To Do's!

This is a simple to-do list application. You can create new to-do lists, add items to these lists, mark items as finished, and more.

How to Use:

  1. Create a New List: Click on "Create new list" in the sidebar to create a new to-do list.
  2. Add Items: After creating a list, you can add new items by clicking on the list and using the input form.
  3. Mark Items as Finished: Check the items you've completed to mark them as finished.
  4. View Lists: Click on "View lists" in the sidebar to see all your to-do lists and their items.

Enjoy staying organized with your to-do lists!
